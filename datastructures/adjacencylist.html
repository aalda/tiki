AdjacencyList

AdjacencyList associates each vertex in the graph with the collection of its neighboring vertices. Internally this is represented as a pair of maps (the graph and its reverse).

The functions provided are:

  • children(v) returns the child vertices of the vertex v.
  • parents(v) returns the parent vertices of the vertex v.

Both functions will return an Option, None will be returned if v does not exist in the graph.

Usage

Constructing an adjacency list

An adjacency list can be constructed as follows:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val adjacencyList = AdjacencyList(List(Edge(1,2),Edge(1,3),Edge(2,3)))
adjacencyList: tiki.AdjacencyList[Int] = tiki.AdjacencyList@5b67f86b

children

Return the child vertices:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val adjacencyList = AdjacencyList(List(Edge(1,2),Edge(1,3),Edge(2,3)))
adjacencyList: tiki.AdjacencyList[Int] = tiki.AdjacencyList@43aaeef8

scala> val xs = adjacencyList.children(1)
xs: tiki.Predef.Option[tiki.Predef.Set[Int]] = Some(Set(2, 3))

parents

Return the parent vertices:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val adjacencyList = AdjacencyList(List(Edge(1,2),Edge(1,3),Edge(2,3)))
adjacencyList: tiki.AdjacencyList[Int] = tiki.AdjacencyList@3ce3879c

scala> val xs = adjacencyList.parents(3)
xs: tiki.Predef.Option[tiki.Predef.Set[Int]] = Some(Set(1, 2))
