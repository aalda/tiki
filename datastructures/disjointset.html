DisjointSet

DisjointSet is a data structure that keeps tracks of non overlapping partitions within a set. The data structure provides the following core functions:

  • find this returns the “representative” (or a ‘parent’) of the element in the set. Thus, two find operations are used to determine if two elements belong to the same partition.
  • union join two partitions together via two elements.
  • components the number of partitions.

Usage

Constructing a disjoint set

A Disjoint-set can be created from a standard set:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val disjoint = DisjointSet(Set[Int](1,2))
disjoint: tiki.DisjointSet[tiki.Predef.Int] = tiki.DisjointSet@2259705

Find

Initially, each element will belong to its own partition:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val disjoint = DisjointSet(Set[Int](1,2))
disjoint: tiki.DisjointSet[tiki.Predef.Int] = tiki.DisjointSet@bfc2a79

scala> disjoint.find(1)
res0: tiki.Predef.Option[tiki.Predef.Int] = Some(1)

Union

Union is used to merge elements of the set:

scala> import tiki._
import tiki._

scala> import tiki.Predef._
import tiki.Predef._

scala> val disjoint = DisjointSet(Set[Int](1,2))
disjoint: tiki.DisjointSet[tiki.Predef.Int] = tiki.DisjointSet@78e18c63

scala> disjoint.union(1,2)
res1: tiki.Predef.Option[tiki.DisjointSet[tiki.Predef.Int]] = Some(tiki.DisjointSet@54db9258)
